QUESTIONS, ANSWERED

Before you go live.

Do viewers install anything?

No. Viewers answer and use commands directly in Kick chat.

Does it control my OBS stream?

No. Emoji Decoder provides Browser Source pages. It never starts streaming or recording.

Can a friend test it safely?

Yes. Streamers can create a temporary guest Test Lab with isolated scores and revocable links.

Will dashboard changes break my OBS links?

No. Themes and game settings update existing Browser Sources automatically.
